# 🚀 S2F Engenharia & Serviços - Institutional Website

This is the repository for the institutional website of **S2F Engenharia & Serviços**, a company specialized in integrated solutions for Electrical Engineering, Mechanical Engineering, and Industrial Maintenance.

The project was developed from scratch focusing on high performance, modern design, and lead conversion for industrial sector clients.

🔗 **[Live Demo (Netlify)](https://s2fengenharia.netlify.app)**

## 📋 About the Project
The website was built to showcase the company's service portfolio, highlight major clients, and facilitate direct contact through multiple channels. The layout is fully responsive, ensuring an excellent browsing experience on both industrial workstations and mobile devices.

## ✨ Key Features
* **Service Catalog:** Interactive cards detailing specialties in Electrical and Mechanical engineering.
* **Client Portfolio:** Optimized gallery featuring partner company logos.
* **Support Channels Integration:**
    * Floating WhatsApp button with native SVG icon.
    * Live Chat widget via **Tawk.to** integrated directly into the HTML.
* **Quote Request Form:** Clean interface designed for lead generation and business inquiries.

## 🛠️ Tech Stack
* **[React](https://reactjs.org/)** - JavaScript library for building user interfaces.
* **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling for fast builds and production optimization.
* **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid and responsive styling.
* **[Netlify](https://www.netlify.com/)** - Hosting and Continuous Deployment (CD).

## 🧠 Lessons Learned & Challenges
In this project, I focused on delivering a professional-grade solution, solving real-world development challenges such as:
1. **Asset Management:** Configured Vite's module import pattern to ensure client logos were correctly bundled and rendered in the Netlify production environment.
2. **Script Integration:** Safely injected third-party scripts (Tawk.to) ensuring they didn't interfere with the React DOM or performance.
3. **UI/UX Optimization:** Refined layout hierarchy and removed CTA redundancies (Call to Action) to create a more intuitive user journey.
